# Weather App using Python

## Project Description

This is a simple weather application built using Python. It allows users to enter a city name and get real-time weather information such as temperature, humidity, wind speed, and weather conditions.

The application uses an API to fetch live weather data and displays it in a clean and user-friendly format.

## Features
 - Get weather information for any city
 - Displays temperature in Celsius
 - Shows weather conditions
 - Displays humidity
 - Shows wind speed
 - Displays atmospheric pressure
 - Allows multiple city searches
 - Handles invalid city input
 - Technologies Used
 - Python
 - Requests library
 - OpenWeather API

## API Used

This project uses the OpenWeather API to fetch real-time weather data.

## How to Run the Project
- Open Google Colab or any Python environment
- Install required library (if not installed):
 
  pip install requests
  
- Copy the project code into your environment
- Replace:

  api_key = "YOUR_API_KEY"

with your actual API key

- Run the program
Enter a city name to get weather details
