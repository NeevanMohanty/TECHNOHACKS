# TECHNOHACKS
## Black Clover: Rise of the Magicless Knight – Text-Based RPG Game
## Project Description

Black Clover: Rise of the Magicless Knight is a text-based role-playing game (RPG) built using Python, inspired by the world of Black Clover.

In this game, the player takes on the role of a determined warrior born without magic, similar to Asta, striving to prove his worth in a kingdom where magic defines status. Along the journey, the player encounters rivals inspired by Yuno and faces challenges that test courage, persistence, and ambition.

The story evolves based on player choices, leading to multiple unique endings.

## Features
- Interactive story with meaningful choices
- Multiple story paths and branching outcomes
- Action-driven gameplay (training, battles, exams)
- Magic Knight entrance exam storyline
- Underdog progression (magicless to powerful warrior)
- Multiple endings based on decisions
- Replayability with different outcomes

## Technologies Used
- Python 3.x
- Standard input/output (console-based interaction)

## How to Play
- Run the Python script
- Enter your name when prompted
- Follow the story and make choices by entering:
   1 or 2
- Explore different story paths and endings
